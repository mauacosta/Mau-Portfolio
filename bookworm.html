Bookworm App

Android Product Design & Development

Academic Project

My Mobile Application Development Course required the creation of an Android App.
I, alongside my teammates Alejandra and Fernando, went beyond it, designing and developing an MVP for the reading-scheduling app, Bookworm.

Definition

Defining Bookworm was not easy, I had an idea of creating an app with the concept of keeping track of the books someone was reading, is reading, or wants to read next. The first approach was in the shape of a moodboard and ideation with the complete team.

Process

Bookworm allows you to search for books and track them as "Want to Read," "Reading," or "Done Reading." With these features in mind, we designed the app to be simple and easy to use while maintaining the brand's style.

Final Product

The end product we developed was an MVP done with Java in Android Studio, that connected with the API from Open Library and showed real-time user data with the help of the Firebase database.
